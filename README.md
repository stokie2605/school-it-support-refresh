# Case Study: Con Ed Website Refresh & Digital Strategy

[cite_start]A comprehensive digital operations, copywriting, and UX strategy built for **Con Ed**, an established IT support provider specializing in primary schools across Hertfordshire[cite: 4, 19]. 

**💰 Project Status:** Completed, Delivered, and Fully Remunerated (£300 Client Brief)

---

## 📋 Project Overview
[cite_start]Con Ed has a highly valuable, relationship-led service model—providing dedicated, weekly on-site technician support to schools—but their previous digital presence buried this unique value proposition under generic, technical IT service lists[cite: 6, 10, 26]. 

[cite_start]This project completely repositions their digital front-door to target non-technical school leaders (Headteachers, Bursars, and School Business Managers) using plain English, clear pain-point resolution, and an outcome-focused approach[cite: 7, 24, 30, 54, 55].

### Key Deliverables:
* [cite_start]**Core Positioning Strategy:** Shifting focus entirely to the premium, relationship-led "weekly on-site technician" model[cite: 8, 11, 41].
* [cite_start]**Homepage Copy Rewrite:** High-conversion, jargon-free copy written for busy school administrators[cite: 55, 75, 166].
* [cite_start]**UX Mock-ups:** A complete structural wireframe detailing the psychological and structural flow of the new homepage[cite: 103, 104, 110].
* [cite_start]**SEO Action Plan:** Targeted local keywords to capture high-intent traffic in the regional education sector[cite: 132, 133].

## 📈 Technical Strategy & Execution
* [cite_start]**Audience Alignment:** Mapped user intents specifically to time-poor, non-technical primary school decision-makers[cite: 55].
* [cite_start]**SEO Foundation:** Integrated high-value local phrases like `IT support for primary schools Hertfordshire` naturally into header structures to improve organic visibility without damaging readability[cite: 134, 141].
* [cite_start]**Grounded, Practical Frameworks:** Intentionally focused on the core service value rather than clouding the strategy with hyped enterprise terminology[cite: 155].
* [cite_start]**AI-Assisted Workflow Efficiency:** Safely utilized LLM tooling to scale competitor research, iterate copy frameworks, and organize structured content blocks, ensuring all final output was manually verified against client constraints[cite: 160, 161, 168].
