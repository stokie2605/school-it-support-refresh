# Case Study: Con Ed Website Refresh & Digital Strategy

A comprehensive digital operations, copywriting, and UX strategy built for **Con Ed**, an established IT support provider specializing in primary schools across Hertfordshire. 

**💰 Project Status:** Completed, Delivered, and Fully Remunerated (£300 Client Brief)

---

## 📋 Project Overview
Con Ed has a highly valuable, relationship-led service model—providing dedicated, weekly on-site technician support to schools—but their previous digital presence buried this unique value proposition under generic, technical IT service lists. 

This project completely repositions their digital front-door to target non-technical school leaders (Headteachers, Bursars, and School Business Managers) using plain English, clear pain-point resolution, and an outcome-focused approach.

### Key Deliverables:
* **Core Positioning Strategy:** Shifting focus entirely to the premium, relationship-led "weekly on-site technician" model.
* **Homepage Copy Rewrite:** High-conversion, jargon-free copy written for busy school administrators.
* **UX Mock-ups:** A complete structural wireframe detailing the psychological and structural flow of the new homepage.
* **SEO Action Plan:** Targeted local keywords to capture high-intent traffic in the regional education sector.

## 📈 Technical Strategy & Execution
* **Audience Alignment:** Mapped user intents specifically to time-poor, non-technical primary school decision-makers.
* **SEO Foundation:** Integrated high-value local phrases like `IT support for primary schools Hertfordshire` naturally into header structures to improve organic visibility without damaging readability.
* **Grounded, Practical Frameworks:** Intentionally focused on the core service value rather than clouding the strategy with hyped enterprise terminology.
* **AI-Assisted Workflow Efficiency:** Safely utilized LLM tooling to scale competitor research, iterate copy frameworks, and organize structured content blocks, ensuring all final output was manually verified against client constraints.
